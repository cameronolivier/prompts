# MCPs
- [Exa Search](https://docs.exa.ai/examples/exa-mcp) - like perplexity but more dev-focused and ranks for recency + relevance
- [Firecrawl](https://www.firecrawl.dev/mcp) - web crawler
- [Pieces](https://docs.pieces.app/products/mcp/get-started) - but also Pieces OS - and App - to manage gained knowledge (of bugs, snippets, etc) and make it available to the LLM.
- [Playwright MCP](https://github.com/microsoft/playwright-mcp)
- [Puppateer](https://www.pulsemcp.com/servers/modelcontextprotocol-puppeteer)
- [Ref.ai](https://ref.tools/) - A documentation MCP that provides concise context-friendly documentation (unlike Context7 which seems to provide _ALL_ docs). [Github repo](https://github.com/ref-tools/ref-tools-mcp)
- [Semgrep](https://semgrep.dev/) - security MCP
- [Sentry](https://docs.sentry.io/product/sentry-mcp/) - Sentry - error and debugging
- [Serena](https://github.com/oraios/serena) - Serena provides essential semantic code retrieval and editing tools that greatly enhance token efficiency.
- [Supabase](https://supabase.com/docs/guides/getting-started/mcp) - MCP for interacting with the Supabase instance for your project
